WP_Notification_plugin by shortcode can put plans in the form of panels.

== Description ==

WP_Notification_plugin allows you to create and configure the notification on your site can be output on the shortcode.

== Installation ==

1. Upload the entire `WP_Notification_plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= How it works? =

Create a notice in the admin menu tab in WP_Notification_plugin. Adjust the color of the notification, the position on the page, shortcode. 
You can add only text of the notification and slag; the other parameters will be set automatically.
In the admin panel you can add new notifications, edit existing. By clicking on the "Copy" button will be automatically generated ready 
shortcode, which you can insert anywhere in a post on the website. When shortcode is copying, you can view the preview. 
When hidden shortcode comes into the visitor visibility, notification will be displayed.
